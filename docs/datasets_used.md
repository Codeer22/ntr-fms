# Datasets Used

## Sentinel-1 (SAR Imagery)
- Provider: European Space Agency (ESA)
- Sensor Type: C-band Synthetic Aperture Radar (SAR)
- Resolution: ~10 meters
- Usage: Forest structure analysis and monitoring under cloud and snow conditions
- Access: Google Earth Engine and SNAP

## Global Forest Watch (GFW)
- Provider: World Resources Institute (WRI)
- Data Type: Forest loss and tree cover change products
- Usage: Validation and cross-checking of detected deforestation events
- Access: Global Forest Watch platform and Google Earth Engine

## SiDroForest (Reference Benchmark)
- Type: Literature-based benchmark
- Usage: Methodological reference for non-tropical forest disturbance analysis
- Note: Used as a reference framework, not as a raw downloadable dataset
