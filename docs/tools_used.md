# Tools and Platforms Used

## Google Earth Engine (GEE)
- Purpose: Large-scale geospatial data processing
- Usage: Multi-temporal analysis of Sentinel-1 and Sentinel-2 imagery
- Benefit: Free, cloud-based, scalable processing

## SNAP (Sentinel Application Platform)
- Provider: European Space Agency (ESA)
- Usage: Sentinel-1 SAR preprocessing (calibration, speckle filtering, terrain correction)
- Benefit: Official and scientifically validated tool

## GitHub
- Purpose: Version control and project documentation
- Usage: Code management, documentation, and progress tracking

## QGIS (Optional)
- Purpose: Visualization and map preparation
- Usage: Final map layout and result presentation (if required)
